TextArea

A textarea.

Demo

Properties

Name Default Description Change
name The name of the field entered into the textarea
type primary, info, success, warning, or danger
rows 4 the visible number of rows to display
value The value entered into the textarea
placeholder Placeholder text to show when the textarea is empty.
maxlength The maximum length of text that can be entered into the textarea
readonly false Set to true to prevent the field from taking input

Code

import { TextArea } from "svelte-toolkit";

<TextArea name="field1" type="" rows="4" placeholder="" maxlength={100} readonly={false} bind:value />
