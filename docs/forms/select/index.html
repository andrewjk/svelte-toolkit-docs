Select

A select.

Demo

Properties

Name Default Description Change
name The name of the field entered into the textbox
type primary, info, success, warning, or danger
value The value entered into the textbox
placeholder Placeholder text to show when the textbox is empty.

Code

import { Select } from "svelte-toolkit";

<Select name="field1" type="" bind:value placeholder="">
  <option value="">-</option>
  {#each countries as country}
    <option>{country.name}</option>
  {/each}
</Select>
