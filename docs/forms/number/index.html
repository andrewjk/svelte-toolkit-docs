Number

A box for entering a number.

Demo

Properties

Name Default Description Change
name The name of the field entered into the number
type info, success, warning, or danger
value 0 The number value
min The minimum allowed number value
max The maximum allowed number value
readonly false Set to true to prevent the field from taking input

Code

import { Number } from "svelte-toolkit";

<Number type="" name="field1" min={null} max={null} readonly={false} bind:value />
