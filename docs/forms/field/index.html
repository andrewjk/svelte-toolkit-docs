Field

A field wraps a form control, providing it with a label, validation etc.

Demo

Properties

Name Default Description Change
label The label to display for the field. A label can also be placed into the named "label" slot
name The name of the data held in this field
value The value held in this field's control
Validation Various validation fields. See validation for info.

Code

<Field label="Your location:" name="location" bind:value>
  <TextBox bind:value />
</Field>
