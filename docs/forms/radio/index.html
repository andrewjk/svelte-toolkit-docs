Svelte Toolkit

Radio

A radio button for selecting a single value from a group.

Demo

Properties

Name Default Description Change
name The name of the group field selected with the radio
type primary, info, success, warning, or danger
group The value of the group field selected with the radio
value The group value to set when this radio is selected
label The text label to display next to the radio

Code

import { Radio } from "svelte-toolkit";

<Radio type="" name="field1" label="Crocodile" value="Crocodile" bind:group />
