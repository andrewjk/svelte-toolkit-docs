Currency

A box for entering a currency, formatted against a specific locale.

Demo

Properties

Name Default Description Change
name The name of the field entered into the currency box
type primary, info, success, warning, or danger
value The currency value
format USD The currency's format, as an ISO currency code (see https://www.currency-iso.org/dam/downloads/lists/list_one.xml)
placeholder Placeholder text to show when the textbox is empty.
min The minimum allowed currency value
max The maximum allowed currency value
readonly false Set to true to prevent the field from taking input

Code

import { Currency } from "svelte-toolkit";

<Currency type="" name="field1" format="USD" placeholder="" min={null} max={null} readonly={false} bind:value />
