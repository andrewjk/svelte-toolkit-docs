File

A file selection control that can be used with drap-and-drop or by selecting a file from a browse dialog.

Demo

Properties

Name Default Description Change
name The name of the field entered into the checkbox
accept The type of files that can be selected, e.g. "audio/*", "video/*", "image/*", or a media type
placeholder Placeholder text to show when no file has been selected.

Code

import { File } from "svelte-toolkit";

<File name="" accept="" placeholder="" />
