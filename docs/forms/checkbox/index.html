Svelte Toolkit

CheckBox

A box that can be ticked.

Demo

Properties

Name Default Description Change
name The name of the field entered into the checkbox
type primary, info, success, warning, or danger
checked Whether the checkbox is checked
group The value of the group field selected with the checkbox when there are multiple checkboxes
value The group value to set when this checkbox is selected and there are multiple checkboxes
label The text label to display next to the checkbox

Code

import { CheckBox } from "svelte-toolkit";

<CheckBox type="" name="field1" label="Please tick the box" bind:checked />
