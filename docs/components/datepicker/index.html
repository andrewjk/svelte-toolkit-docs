Date Picker

A dropdown date picker that can be used to enter or select a date.

Demo

Properties

Name Default Description Change
value the selected value Fri Aug 13 2021 12:36:27 GMT+1000 (Australian Eastern Standard Time)
placeholder Placeholder text to show when the input is empty.
dateFormat d MMM yyyy the format to display the date in, where 'd' is days, 'M' is months and 'y' is years
dateOrder mdy the order to process dates entered by the user (e.g. 31/1 vs 1/31)

Code

import { DatePicker } from "svelte-toolkit";

<DatePicker value={Fri Aug 13 2021 12:36:27 GMT+1000 (Australian Eastern Standard Time)} placeholder="" dateFormat="d MMM yyyy" dateOrder="mdy" />
