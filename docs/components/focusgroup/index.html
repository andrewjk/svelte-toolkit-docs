Svelte Toolkit

Focus Group

A list of items that is focused as a whole. The user can use the left and right arrow keys to move between the items.

Demo

Properties

Name Default Description Change
tagName button the tag of the item to focus e.g. button, a, etc

Code

import { FocusGroup, FocusGroupItem } from "svelte-toolkit";

<FocusGroup tagName="button">
  {#each items as item}
    <FocusGroupItem>
      <ImageButton>{item}</ImageButton>
    </FocusGroupItem>
  {/each}
</FocusGroup>
