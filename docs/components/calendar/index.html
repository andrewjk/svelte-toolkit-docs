Properties

Name Default Description Change
startOfWeek 1 the day of the week to start with, where 0 = Sunday, 1 = Monday etc
selectable false set to true to allow the user to select a day
value the value of the selected date (if selectable is true) TODO:
events an array of events to display in the calendar, where each event can have the properties date, content and color TODO:

Code

<Calendar startOfWeek={1} selectable={false} value={new Date()} {events} />
