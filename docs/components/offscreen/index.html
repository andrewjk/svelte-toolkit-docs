Off-Screen

A panel that can appear from off-screen.

Demo

Properties

Name Default Description Change
visible false true to show the off-screen
position left left, top, right, or bottom

Code

import { OffScreen } from "svelte-toolkit";

<OffScreen visible={false} position="left">
  Off-screen content
</OffScreen>
