Badge

A badge for displaying information such as an unread count.

Demo

Notifications 5

Properties

Name Default Description Change
type info info, success, warning, or danger
content The content to display in the badge, which can also come from slot content

Code

import { Badge } from "svelte-toolkit";

Notifications <Badge type="info">5</Badge>

~ OR ~

Notifications <Badge content="5" type="info" />
