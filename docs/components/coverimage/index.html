Svelte Toolkit

Cover Image

A div with a background-image, which allows you to do some things you can't do with a simple img.

Demo

Properties

Name Default Description Change
src The image source as a URL
position 50% 50% Overrides the background-position style
repeat no-repeat Overrides the background-repeat style
size cover Overrides the background-size style
height 0 You MUST provide a height for your image, either via this property or by CSS styling
width 100% You can also provide a width for your image, either via this property or by CSS styling
alt Alternative text used to describe this image to a screen-reader

Code

import { CoverImage } from "svelte-toolkit";

<CoverImage src="/images/image1.jpg" position="" repeat="" size="" height="100px" width="" alt="A kangaroo" />
