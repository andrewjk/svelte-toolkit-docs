Svelte Toolkit

Tag

A tag for displaying information about an item.

Demo

Awesome! Great Job!

Properties

Name Default Description Change
type info primary, info, success, warning, or danger
content The content to display in the tag, which can also come from slot content
closable false Set to true to display a close button at the right of the tag

Code

import { Tag } from "svelte-toolkit";

<Tag type="info" closable={false}>Awesome!</Tag>

~ OR ~

<Tag content="Awesome!" type="info" closable={false} />
