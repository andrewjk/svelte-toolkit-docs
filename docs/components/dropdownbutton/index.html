Svelte Toolkit

Drop Down Button

A drop-down button menu for selecting.

Demo

Properties

Name Default Description Change
buttonType info primary, info, success, warning, or danger
buttonSize medium small, medium, large, or inline
position below below, above
alignment start start, center, end

Slots

Name Description Change
element Replace the button with components of your choice.

Code

import { DropDownButton, DropDownMenuItem } from "svelte-toolkit";

<DropDownButton buttonType="info" buttonSize="medium" position="below" alignment="start">
  <span>Items...</span>
  <div slot="menu">
    {#each items as item}
      <DropDownMenuItem>
        <a href={item.href}>{item.name}</a>
      </DropDownMenuItem>
    {/each}
  </div>
</DropDownButton>
