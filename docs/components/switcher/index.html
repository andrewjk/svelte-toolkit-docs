Switcher

A set of panels that can be switched between, either programmatically or using one of the navigation components.

Demo

Kangaroo

It jumps

Platypus

It swims

Kookaburra

It flies

Properties

Name Default Description Change
index 0 the index of the active item

Events

Name Args
hanged index: the index of the active item
activeChange (SwitcherItem) active: whether the changed item is active

Code

<Switcher index={0}>
  {#each $items as item}
    <SwitcherItem>
      <h4>{item.header}</h4>
      <p>{item.text}</p>
    </SwitcherItem>
  {/each}
</Switcher>
