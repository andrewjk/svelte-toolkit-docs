Svelte Toolkit

Palette

A palette that you can use to choose a color.

Demo

Properties

Name Default Description Change
value the value of the color that is selected, which can be hex, rgb() or a named CSS color
standardColors an array of generic standard colors you can pass in an array of custom standard colors that the user can select

Code

import { Palette } from "svelte-toolkit";

<Palette value="green"
