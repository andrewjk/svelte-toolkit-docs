Properties

Name Default Description Change
type info info, success, warning, danger, select, cancel
size small, medium, large, or inline
submit false Set to true to make this a submit button
reset false Set to true to make this a reset button
disabled false Set to true to disable the button
loading false Set to true to replace the button's text with a loading animation
