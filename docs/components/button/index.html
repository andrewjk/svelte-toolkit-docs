Button

A button, per https://www.w3.org/TR/wai-aria-practices-1.1/#button .

Colors

Sizes

Demo


Properties

Name Default Description Change
type primary primary, info, success, warning, danger, cancel
size medium small, medium, large, or inline
submit false Set to true to make this a submit button
reset false Set to true to make this a reset button
disabled false Set to true to disable the button
loading false Set to true to replace the button's text with a loading animation
rounded false Set to true for rounded borders
outlined false Set to true for an outlined button

Code

import { Button } from "svelte-toolkit";

<Button type="primary" size="medium" submit={false} reset={false} disabled={false} loading={false} rounded={false} outlined={false}>
  Click me
</Button>
