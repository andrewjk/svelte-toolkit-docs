Svelte Toolkit

Drop Down Link

A drop-down link menu for selecting.

Demo

Properties

Name Default Description Change
position below below, above
alignment start start, center, end

Slots

Name Description Change
element Replace the button with components of your choice.

Code

import { DropDownLink, DropDownMenuItem } from "svelte-toolkit";

<DropDownLink position="below" alignment="start">
  <span>Items...</span>
  <div slot="menu">
    {#each items as item}
      <DropDownMenuItem>
        <a href={item.href}>{item.name}</a>
      </DropDownMenuItem>
    {/each}
  </div>
</DropDownLink>
