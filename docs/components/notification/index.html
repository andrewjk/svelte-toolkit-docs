Notification

A notification that can be shown to the user.

Demo

You might be interested to know this...

Properties

Name Default Description Change
type info primary, info, success, warning, or danger
content The content to display in the notification, which can also come from slot content
closable false Set to true to display a close button at the right of the notification
duration -1 The number of milliseconds to wait before closing the notification. Set to -1 to never close
solid true Set to true to make the notification have a solid background

Events

Name Args
closed

Code

import { Notification } from "svelte-toolkit";

<Notification content="You might be interested to know this..." type="info" closable={true} duration={0} solid={false} />

~ OR ~

<Notification type="info" closable={true} duration={0} solid={false}>
  You might be interested to know this...
</Notification>
