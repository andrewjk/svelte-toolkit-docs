Bar Chart

A simple SVG bar chart for displaying data.

Demo

Properties

Name Default Description Change
width 400 The width of the chart, in pixels
height 200 The height of the chart, in pixels
BarChartItem: value The value of the item
BarChartItem: label The label of the item to show along bottom of the chart
BarChartItem: color A default color, chosen from a list The color of the item

Code

import { BarChart, BarChartItem } from "svelte-toolkit";

<BarChart width={400} height={200}>
  <BarChartItem value="25" label="Item 1"></BarChartItem>
  <BarChartItem value="35" label="Item 2"></BarChartItem>
  <BarChartItem value="40" label="Item 3"></BarChartItem>
</BarChart>
