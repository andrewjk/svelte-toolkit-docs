Pie Chart

A simple SVG pie chart for displaying data.

Demo

Properties

Name Default Description Change
size 200 The size of the chart, in pixels
radius 100 The size of the chart's radius, in pixels
PieChartItem: value The value of the item
PieChartItem: color A default color, chosen from a list The color of the item

Code

<PieChart size={200} radius={100}>
  <PieChartItem value="25"></PieChartItem>
  <PieChartItem value="35"></PieChartItem>
  <PieChartItem value="40"></PieChartItem>
</PieChart>
