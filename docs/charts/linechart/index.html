Line Chart

A simple SVG line chart for displaying data.

Demo

Properties

Name Default Description Change
width 400 The width of the chart, in pixels
height 200 The height of the chart, in pixels
LineChartSeries: value [] The values of the items
LineChartSeries: label [] The labels of the items to show along bottom of the chart
LineChartSeries: color A default color, chosen from a list The color of the series

Code

<LineChart width={400} height={200}>
  <LineChartSeries
    values={[25, 35, 40]}
    labels={['Item 1', 'Item 2', 'Item 3']} />
</LineChart>
