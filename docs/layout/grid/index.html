Properties

Name Default Description Change
spaced false Whether there should be space between the rows and columns of the grid.
cols 12 The number of columns in the grid (up to 12).
smcols The number of columns in the grid on a small screen (and up).
mdcols The number of columns in the grid on a medium screen (and up).
lgcols The number of columns in the grid on a large screen (and up).
xlcols The number of columns in the grid on an extra large screen (and up).
GridItem span 1 The number of columns to span with a GridItem (up to 12).
GridItem smspan The number of columns to span with a GridItem on a small screen (and up).
GridItem mdspan The number of columns to span with a GridItem on a medium screen (and up).
GridItem lgspan The number of columns to span with a GridItem on a large screen (and up).
GridItem xlspan The number of columns to span with a GridItem on an extra large screen (and up).

Code

import { Grid, GridItem } from "svelte-toolkit";

<Grid
    spaced={false}
    cols="6"
    smcols="0"
    mdcols="0"
    lgcols="0"
    xlcols="0">
  <GridItem
      span="1"
      smspan="0"
      mdspan="0"
      lgspan="0"
      xlspan="0">
    1
  </GridItem>
  <GridItem>2</GridItem>
  <GridItem>3</GridItem>
  <GridItem>...</GridItem>
</Grid>
