Stacks allow you to lay items out vertically or horizontally.

Properties

Name Default Description Change
direction vertical vertical, horizontal
StackItem size auto grow, auto

Code

import { Stack, StackItem } from "svelte-toolkit";

<Stack direction="vertical">
  <StackItem size="auto">
    Item 1
  </StackItem>
  <StackItem>Item 2</StackItem>
  <StackItem>Item 3</StackItem>
  <StackItem>...</StackItem>
</Stack>
